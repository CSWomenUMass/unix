# unix
Materials for UNIX Workshop

# Topics 
- File manipulation: `cat`, `head`, `tail`, `less`, `diff`,  
- Directory navigate: `pwd`, `cd`, `cd ..`, `cd -`, 
- Permission: `chmod` 
- Network utility: `ping`, `traceroute`, `ifconfig`, `wget`, `curl`, 
- Locate files: `locate`, `which`, `whereis`, `find`
